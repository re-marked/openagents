# 02 — Product Specification

## Product Overview

OpenAgents has three distinct surfaces:

### 1. Public (openagents.com)
The storefront. Landing page, discovery, search, Agent previews. No account required to browse.

### 2. Workspace (openagents.com/workspace)
The personal dashboard. Your Projects, Teams of Agents, chat interface, settings, usage. Requires login.

### 3. Platform (platform.openagents.com)
The creator studio. Publish Agents, analytics, payouts, YAML editor. For developers. Requires GitHub login.
